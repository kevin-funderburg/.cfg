My Dot files
==============

Repo to store the various dot files for my system so they can be synced across my devices. Methodology based on this [article on Atlassian](https://www.atlassian.com/git/tutorials/dotfiles)

_______

Files
--------
- Files stored:
   - .zshrc
   - .bashrc
   - .vimrc
   - .gitconfig
   - .gitignore_global
